Dépôt  local:
Il s'agit des fichiers stockés dans votre ordinateur.

Dépôt en ligne:
Il s'agit des fichiers stockés dans des datacenters.

Git:
Git est un gestionnaire de versions.

Github: 
GitHub est un service en ligne qui héberge les dépôts Git

Initialisation d'un fichier:
git init

Cloner un fichier en ssh:
git clone git@github.com:user_github/nom_branche

Indexer vos fichiers:
git add

Créez une nouvelle version :
git commit 

Créez une nouvelle version avec commentaire  :
git commit -m "votre commentaire"

Créez une nouvelle version avec commentaire plus validation des commits :
git commit -S -m "votre commentaire"

Pour configurer clé gpg:
https://docs.github.com/en/authentication/managing-commit-signature-verification/about-commit-signature-verification


Pour configurer clé ssh:
https://docs.github.com/en/authentication/connecting-to-github-with-ssh/about-ssh

Reliez votre dépôt local au dépôt distant:
git  remote add https://github.com/user_github/nom_projet

Mettre les modifications en ligne:
git push
