Voir les différences entre la version git et la vôtre
git diff

Affichage des commits sur le repository
git log
les différentes options:
affiche les commmits de toutes les branches.
git log--all

affiche les titres des commits
git log --oneline

affiche  un graph avec les branches
git log --graph

affiche le résumé des changments de chaque commit ave --stat
git log -all --oneline --stat
